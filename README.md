# Machine Learning Based Classification of Healthy, Asthma, and COPD Patients

This project presents a machine learning approach for classifying patients into Healthy, Asthma, and COPD categories using clinical and spirometry features.

The project implements and compares four supervised machine learning algorithms:

- Logistic Regression
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Features
- Data preprocessing and feature engineering
- Label encoding and feature scaling
- Model training and evaluation
- Accuracy, F1-Score, and ROC-AUC analysis
- Confusion Matrix visualization
- Feature importance analysis

## Technologies Used
- Python
- Google Colab
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
